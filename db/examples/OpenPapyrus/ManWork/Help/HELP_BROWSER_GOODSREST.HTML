Таблица товарных остатков



Остатки товаров

Типичный вид таблицы товарных остатков показан на рис..
Здесь видны следующие колонки:
Наименование товара Полное наименование товара
Остаток Колонка разбита на две графы: остаток в торговых единицах и остаток в физических единицах. Если для товара не задано соотношение физических единиц к торговым, то значение остатка в физических единицах будет пустым. Если для товара ведется независимый учет в торговых и физических единицах то, разумеется, величины в этих двух графах не будут связаны простым отношением.
Цена Эта колонка содержит две графы: учетная цена поступления товаров и учетная цена реализации товаров. Важно помнить, что значения в этих графах зависят от опции расчета цены, установленных в фильтре. Если пользователь не имеет прав доступа к ценам поступления, то колонка цены поступления будет пустой.
Сумма Две графы, представляющие простое произведение колонки Остаток в торговых единицах на соответствующую графу колонки Цена. Если пользователь не имеет прав доступа к ценам поступления, то колонка суммы в ценах поступления будет пустой.
Наценка Процент наценки, вычисленный по двум предыдущим графам следующим образом: $100 * \frac{P - C}{C}$, где: \\$C$ - сумма в ценах поступления % \\$P$ - сумма в ценах реализации.

Если отчет строился с флагом Раздельно по каждому складу, то в таблице будет присутствовать колонка наименования склада (для каждой пары товар-склад, для которой есть значимые результаты отчета, будет показана отдельная строка).
Если, в дополнение к этому флагу, установлен Кросстаб, то таблица будет выглядеть по иному: первая колонка отображает наименования товаров, верхняя строка - наименования складов, а в пересечениях будут показаны остатки товаров на складах в торговых единицах.

Действия над таблицей товарных остатков


Лоты Enter Показывает таблицу лотов по текущему товару. Заказы на товары F3 Показывает таблицу заказов по текущему товару. Товар F2 Позволяет просмотреть и изменить информацию по текущему товару. Положить в корзину Space Добавляет текущий товар в корзину. Добавить в корзину все товары из отчета Ctrl-F5 Переносит в выбранную корзину все товары из отчета. Будьте внимательны с этой функцией. Если отчет очень большой, то операция внесения всех товаров из него в корзину может занять продолжительное время, к тому же работать с такой корзиной будет крайне затруднительно. Группировка операций по товару F5 Показывает отчет группировки товарных операций по выбранному товару. Предварительно запрашивается период, за который следует рассчитать группировку товарных операций. При последующих обращениях к этой функции из данной таблицы введенный период будет сохраняться. Прогноз Ctrl-F9 Рассчитывает прогноз продаж по выбранному товару за период, который следует ввести в появившемся диалоге. Прогноз рассчитывается на основании таблицы продаж. Таблица продажи товара Ctrl-S Показывает таблицу продаж по выбранному товару. Анализ товарных операций Ctrl-F3 Строит отчет анализа товарных операций по выбранному товару с группировкой по датам из весь период, доступный в базе данных. Установить контрактные цены Ctrl-P Специальная функция, позволяющая установить контрактные цены для поставщика, по которому рассчитан отчет (если поставщик не выбран, то функция не работает). Метод работы этой функции таков: перебираются все товары из отчета и для каждого из товаров и по каждому складу, входящему в список фильтра, устанавливается значение контрактной цены, равное цене поступления из отчета. Главное назначение функции - облегчить первоначальную установку контрактных цен на начальном этапе формализации ценовых договоренностей с поставщиками. Котировки Ctrl-Q Сервисная функция, позволяющая просмотреть котировки по текущему товару. Контрактные цены поставщиков Ctrl-W Вызов этой функции открывает диалог просмотра и редактирования контрактных цен на выбранный товар по поставщикам. Товарная матрица Ctrl-M Открывает диалог просмотра и редактирования товарной матрицы выбранного товара. Печать F7 См. ниже Итоги F9 Показывает итоговые значения таблицы остатков товаров

Печать отчета



Печать остатков товаров

Функция печати позволяет распечатать отчет с выбором варианта отчета. Предварительно вы увидите диалог, показанный на рис.. В диалоге присутствуют следующие управляющие элементы:
Переключатель вида отчета Позволяет выбрать один из следующих вариантов отчета:
В ценах поступления и реализации Результирующий отчет будет содержать цены поступления и реализации товаров, а также их суммы.
В ценах поступления Результирующий отчет будет содержать только цены поступления товаров и суммы в ценах поступления.
В ценах реализации Результирующий отчет будет содержать только цены реализации товаров и суммы в ценах реализации.
Инвентарная ведомость Будет распечатана ведомость для проведения инвентаризации на складе. Если вы выберите инвентарную ведомость без остатков, то в колонке Учетный остаток не будут проставляться соответствующие значения. Для более подробной информации обратитесь к разделу, посвященному проведению инвентаризации товаров.
Бланк учета продаж на складе Эта форма полезна при проведении регулярной инвентаризации на удаленном складе, не обеспеченном компьютеризованной системой учета, для последующего занесения разницы между учетным и фактическим остатками как продаж на удаленном складе.
Специальный отчет Этот вариант предназначен для печати заказного специализированного отчета и здесь не рассматривается.
Переключатель сортировки отчета Позволяет выбрать сортировку строк печатного отчета.
Печатать штрихкоды При включенном флажке будет напечатан отчет, содержащий колонку штрихкодов.
Остатки в упаковках Если этот флажок включен, то величины остатков будут печататься в формате количество упаковок / емкость упаковки + остаток. Например: 5/24+1.
Не группировать Подавляет группировку отчета по наименованиям групп товаров.

Поиск в таблице товарных остатков


Опция поиска в таблице товарных остатков несколько модифицирована по сравнению со стандартной функцией поиска в большинстве остальных таблиц. Если вы нажимаете комбинации клавиш Ctrl-F и Ctrl-G, то выполняется стандартный поиск.
То же самое происходит, если вы нажали клавишу, соответствующую какой-либо букве или звездочку (*).
Если же вы нажали клавишу, соответствующую цифре, то появляется окно ввода штрихкода для поиска строки с товаром, имеющим введенный штрихкод.

Быстрый выбор товарной группы


В панели инструментов таблицы товарных остатков представлен комбо-бокс, позволяющий быстро переключить отчет на какую-либо товарную группу. При этом остальные параметры фильтрации остаются неизменными.
