Диалог критерия значения тэга

На рис. показан диалог одного критерия фильтра значений тэгов.

Тэг Тэг, для которого указывается критерий фильтрации.
Подсказка Справочный текст, подсказывающий формат ввода ограничений в зависимости от типа значения тэга.
Ограничение Текстовое поле, в котором вводится ограничение или отображается выбранный из списка критерий.
... Для перечисляемых тэгов и тэгов, ссылающихся на другие объекты данных, по этой кнопке можно выбрать значение, которым должно присутствовать в тэге объекта, дабы он удовлетворял фильтру.
Выбор специальных критериев Блок радио-кнопок, позволяющий выбрать специальный критерий фильтрации по тэгу:
Ограничение При этом варианте работает обыкновенный критерий выбора тэга по введенному в поле Ограничение ограничению.
Пустой Этот вариант предприсывает системе считать условие удовлетворенным, если тэг по объекту имеет пустое значение либо не имеет тэга заданного типа вообще.
Существующий При этом варианте условие считается удовлетворенным, если объект имеет тэг заданного типа не зависимо от значения этого тэга.
Цвет Цвет применяется для правил индикации. Выбранный здесь цвет будет применяться для выделения объектов, удовлетворяющих данному критерию.
