Таблица товарного отчета по операции





Наиболее общий вид таблицы товарного отчета по операции показан на рис.. Если в качестве анализируемой операции вы выбрали переоценку или использовали флаг Разница цен при межскладской передаче, то таблица будет иметь несколько иной вид.
Колонка Количество в физических единицах содержит правильное значение только в случае, если для товаров указаны корректные соотношения физических единиц к торговым.
Колонка Сумма в ценах реализации содержит сумму чистой цены реализации по операции, то есть Номинальная цена реализации - Скидка.
Колонка Доход рассчитывается только для операций, имеющих признак Доходная. При этом значение в этой колонке равно разнице между суммой в чистых ценах реализации (с учетом скидки) и суммой в ценах поступления для доходных операций.

Действия с таблицей товарного отчета по операции


Лоты Enter Просмотр лотов по товару, соответствующему выбранной строке отчета Лоты заказов Shift-F3 По этой команде доступен список заказов на выбранную товарную позицию. Товар F2 Просмотр и редактирование товара, соответствующего выбранной строке. Если к отчету применена подстановка товара, то эта функция показывает подставленный вместо товара объект. Например, если используется подстановка поставщика, то будет показан поставщик, которому соответствует текущая % строка таблицы. Анализ товарных операций F3 Эта команда позволяет рассчитать анализ товарных операций по выбранной позиции. При выборе этой команды вы увидите фильтр анализа товарных операций в котором установлен период такой же, как и в товарном отчете по операции и текущий товар из таблицы. Остальные параметры фильтрации вы можете изменить самостоятельно. Пока вы находитесь в таблице отчета, система хранит условия последнего выбранного анализа товарных операций с тем, чтобы вы могли и по другим товарным позициям посмотреть аналогичный отчет. Группировка операций F5 Группировка операций по товару, соответствующему выбранной строке отчета, за период и по складу, выбранным в фильтре товарного отчета по операции Положить в корзину Ctrl-F4 Положить выбранный товар в корзину Добавить в корзину все товары из отчета Ctrl-F8 Загружает в выбранную корзину все номенклатурные позиции, вошедшие в отчет. Обратите внимание на то, что обычно отчет содержит весьма обширный список товаров. По этому, внесение этого списка в корзину и работа с такой корзиной могут занимать продолжительное время. Сортировка Ctrl-O Сортировка отчета по одному из следующих критериев:
  • По наименованию товара
  • По количеству товара в строке
  • По сумме в ценах поступления
  • По сумме в ценах реализации
  • По доходности
Используя возможность сортировки, вы можете эффективно анализировать свой товарный ассортимент. Печать F7 Отправка отчета на печать или экспорт в какой-либо из доступных форматов Итоги F9 Просмотр итогов по отчету
