A.Импорт справочника кассиров из Кристалла в текстовый файл
-----------------------------------------------------------

1. На компьютере, где установлен Кристалл:
   Пуск -> Программы -> Microsoft SQL Server -> Import and Export Data

2. "Далее" -> выбрать Database - SES

3. "Далее" -> выбрать Destination - Text File,
                      File Name   - Путь к каталогу\cashiers.txt

4. "Далее" -> выбрать Use a query to specify the data to transfer

5. "Далее" -> Query Builder -> Source Tables, выбрать SES.Cashiers:

    TabNumber    >
    CashierName  >
    Passw        >
    Rights       >
               
6. "Далее" -> выбрать >>, после этого все время "Далее"


B. Настройка кассиров в PAPYRUS
-------------------------------

1. Завести новый вид персоналии для кассиров (например,"Кассиры").

2. Завести новый тип регистра для табельного номера (например, "Табельный номер"),
   указать набор символов для этого регистра (например, "tabnum"),
   установить флаг "Неуникальный номер".

3. В конфигурации оборудования выбрать вид персоналии - кассиры.

4. В PP.INI в секции CONFIG прописать CashierTabNumSymb=tabnum.

C. Загрузка кассиров в PAPYRUS
------------------------------

1. Скопировать файл cashiers.txt в каталог, откуда в PAPYRUS
   импортируются данные из Кристалла.

2. При закрытии сессии справочник кассиров будет загружен в PAPYRUS,
   а текстовый файл будет удален.
   После этого администрирование кассиров вести только в PAPYRUSе.
