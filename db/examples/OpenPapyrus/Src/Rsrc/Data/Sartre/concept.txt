// Набор концепций для загрузки в семантическую базу данных
// UTF-8 (65001)
//
// Символьные обозначения понятия обозначаются строчными буквами с префиксом ':'
// Наименования для обозначения понятий берутся из латинского и английского языков.
// Обобщенные понятия имеют приставку 'gener_'
//
// Унифицированные обозначения понятий не должны иметь пробелов. В качестве разделителей
// можно использовать символы '_' и '-'.
//
// Если для какого либо понятия существует международный стандартизованный символ, то
// преимущественно, такие понятия обознаются этим символом с некоторым префиксом.
// Например :ling_ru - русский язык (префикс ling и ISO-символ русского языка ru).
//
// Текст в кавычках трактуется как одно слово. Например "пр-д", "с/м" "д.т.н." 
// Это правило введено изначально ради сокращений, но, вероятно, будет полезно и в других случаях.
// Экранирование внутренних кавычек пока не предусмотрено, поэтому не следует использовать слова, содержащие кавычки. 
// Потом - видно будет.
// Важно: крайне не желательно вводить лексемы в кавычках, содержащие пробелы или что-то еще, что может навредить 
// последующему поиску соответствия в базе данных.
//
// Конструкции 
//   1) ":concept_symb1:concep_symb2" - концепция с символом concept_symb1 является экземпляром концепции concept_symb2
//   2) ":concept_symb1::concep_symb2" - концепция с символом concept_symb1 является подклассом концепции concept_symb2
//   3) "concept_text:concep_symb2" - текст concept_text представляет n-gram который фиксируется как анонимная концепция, являющаяся экземпляром концепции concept_symb2
//   4) "concept_text::concep_symb2" - текст concept_text представляет n-gram который фиксируется как анонимная концепция, являющаяся подклассом концепции concept_symb2
//   5) "[lang]concept_text:concep_symb2" - то же, что и (3) но с уточнением языка
//   6) "[lang]concept_text::concep_symb2" - то же, что и (4) но с уточнением языка
//   7) "concept_text=:concep_symb2" - текст concept_text представляет n-gram который является естественно-языковым обозначением концепции concept_symb2
//   8) "[lang]concept_text=:concep_symb2" - то же, что и (7) но с уточнением языка
//
// Порядковые номера (индексы перечислений) всегда начинаются с 1 (не с 0 как в c++).
//

//
// Зарезервированные свойства концепций
