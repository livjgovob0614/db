
	КРАТКАЯ СПРАВКА ПО РАБОТЕ С СИСТЕМОЙ PAPYRUS

Запуск
	ppstart.bat

Выход
	Закрыть все окна и выбрать пункт меню Справочники|Выход
	или нажать Alt-X ('X' - английския).

Диалог товарного документа

	Ctrl-Enter	Аналогично кнопке <Строки документа...>
	F7		Печать документа

	Поле 'скидка' заполняется тогда, когда необходимо ввести скидку
	на весь документ. Для этого отметьте флажок "Скидка на весь документ"
	при этом поле скидки станет доступным для ввода. Затем в этом поле
	проставте либо сумму скидки, либо количество процентов скидки,
	завершенное символом '%' или '/'. Например, для ввода скидки 100 руб
	следует просто указать в поле ввода скидки "100", но для
	скидки в 10% следует в поле скидки ввести "10%" или "10/".
	Заданная суммарная скидка будет распределена по всем товарным строкам
	пропорционально. При этом может получиться ситуация, когда
	результирующая общая скидка будет немного отличаться от той, что
	введена изначально. Это связано с проблемой округления.

	Если необходимо снять с цен реализации товаров, вбитых в документ
	сумму акциза, то в поле общей скидки необходимо набрать букву 'A'
	(русскую либо латинскую в любом регистре). Если в общей конфигурации
	задан флаг "Цены не включают акциз", то это приведет к обратному 
	эффекту (акциз будет добавлен к ценам реализации).
	Например в поле общей скидки можно поставить следующие значения:
		10%    Общая скидка 10%
		100    Общая скидка 100 руб.
		A      Снять (добавить) акциз с цен товаров
		10%A   Снять (добавить) акциз и с полученной суммы еще дать
		       скидку 10%

Броузер товарных строк документа

	Enter		Редактировать строку
	Insert		Стандартное добавление строки
	Delete		Удалить строку
	F2		Расширенное добавление строки
	F3		Поиск товара для добавления строки по штрих-коду
	F6		Добавление строки для продажи по заказу
	F9              Автоматическое добавление товарных строк по 
	                частичным структурам
	Клавиши 0..9,'*'	Поиск товара для добавления строки по штрих-коду

	При добавлении строки в привязанный документ (возврат)
	в верхней части экрана появляется броузер строк документа к которому
	привязывается текущий документ. В этом вспомогательном броузере
	строку можно выбрать нажатием Enter. Если требуется вставить в
	текущий документ все (которые возможно) строки из связанного документа,
	то необходимо нажать Ctrl-Enter.

	При добавлении строки по заказу (F6) сначала появляется список
	документов заказов, из которых можно выбрать товар, и, после выбора
	документа, появляется броузер строк, из которого следует выбрать
	нужную (Enter) или все возможные (Ctrl-Enter).

Поиск товаров по штрих-коду

	Во всех диалогах, где присутствуют комбинированные списки выбора
	товара (фильтр по лотам, фильтры карточки товара, группировки товарных
	операций и т.д.), а также в броузере товарных строк документа
	товар можно выбрать по штрих-коду. Для этого, сделав поле выбора
	товара активным, необходимо нажать F2 (в броузере товарных строк
	документа - F3 либо цифровую клавишу или '*'). Вслед за этим на
	экране появится небольшое окно для ввода штрих-кода. В поле
	этого окна следует ввести искомый штрих-код полностью, либо, если
	требуется найти все штрих-коды, содержащие некоторую
	последовательность цифр, предварить эту последовательность звездочкой
	('*'). Затем нажать Enter для поиска или Escape если передумали искать.
	Если был задан поиск со звездочкой и существуют штрих-коды, содержащие
	заданную строку, то на экране появится список найденных кодов с
	наименованиями товаров. Выберите интересующий вас товар и нажмите Enter.
	При поиске по точному соответствию если штрих-код найден, то
	произойдет то, что должно произойти при выборе соответсвующего товара,
	если штрих-код не найден, то в большинстве случаев вообще ничего не
	произойдет.

	Примеры:
		"0288822" - Сигареты Мальборо
		"*460114" - Все товары, коды которых содержат "460114"
			460114000001
			460114000122
			890460114001
			и и.д.

Диалог редактирования строки товарного документа

	F3		Установка цены реализации по котировки товара 
			(только расходные операции).
			Если была найдена подходящая котировка, то цена реализации
			устанавливается равной найденной котировке. После этого
			поля цены и скидки становятся недоступными, и общая скидка
			по документу не изменяет цену по этой позиции.
	Shift-F3	Снятие зависимости цены реализации от котировки. Отменяются
			ограничения, описанные в предыдущем параграфе, но цена остается
			равной заданной до этого котировке.
	F2		Если текущим является поле цены поступления, либо цены
			реализации, то вызывается специализированный калькулятор для
			расчета цены реализации по проценту наценки на цену поступления.
