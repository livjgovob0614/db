Настройки в весах BIZERBA для работы с PAPYRUS
----------------------------------------------

1. В настройках TCP/IP порт 1025.

2. Настройки штрихкода:
   - Структура внутреннего кода - 2х (х=0-9);
   - Тип внутреннего кода - {UPC/EAN 13};
   - Код. значение - {Вес};
   - Код. разряды - {5};
   - Проверочная цифра - [ ];
   - Опция - {Взвешивание х 1,0};
   - Условное число - 2х (как в структуре внутреннего кода);
   - Содержание кода - ХХХХХ_____


Настройки в PAPYRUS для работы с BIZERBA
----------------------------------------

1. Админ-Прочие конфигурации-Конфигурация справочника товаров:
   - Префикс кода весового товара - 2х (как в структуре внутреннего кода).

2. Справочники-Оборудование-Весы:
   - IP-адрес - как в весах;
   - Флаг "Использовать TCP/IP протокол" установить.

3. bizsc.dll должна быть зарегистрирована в системном реестре:
   regsvr32.exe bizsc.dll

4. В случае, если bizsc.dll будет разрешать передачу 5 знаков в штрихкоде:
   в pp.ini в секции [system] указать параметр BizerbaWholeBarcode=1.
