При настройке модуля для работы с весами DIGI выбрать следующие параметры:
1. Маска файла импорта данных - dgimp*.dat или dgimp???.dat.
2. Директория файлов импорта должна совпадать с путями экспорта в Papyrus.
3. Кодировка шрифта - Windows (WIN1251).
4. Формат файлов импорта - простой.
5. Последний байт IP-адреса весов совпадает с логическим номером весов в Papyrus.
