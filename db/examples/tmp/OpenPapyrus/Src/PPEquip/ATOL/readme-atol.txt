Настройки РМК Атол для совместной работы с PAPYRUS
--------------------------------------------------
1. При установке РМК на кассах путь к БД указать, как путь к БД на сервере.
2. При установке РМК на сервере в настройках на закладке "Налоги" добавить:
   - налоговую группу НДС 10%, код 1000 с налоговой ставкой НДС 10%;
   - налоговую группу НДС 18%, код 1800 с налоговой ставкой НДС 18%.
При изменении налоговых ставок по аналогии - для ставки AB.CD% добавить
налоговую группу с кодом ABCD с налоговой ставкой AB.CD%.
3. При установке РМК на сервере в настройках на закладке "Оплата"
   код для наличных - 1, другие коды воспринимаются, как безнал.
4. При установке РМК на сервере в настройках на закладке "Обмен данными":
   1) Обмен данными - обмен через текстовые файлы;
   2) Флаг "Загружать товары при не выгруженных продажах" установить;
   3) Файл загрузки - путь к файлу загрузки\atolimp.txt;
   4) Файл выгрузки - путь к файлу выгрузки\atolexp.txt;
   5) Автоматический обмен данными - флаги "загрузка" и "выгрузка" установить;
   6) Служебный файл-флаг для загрузки - путь к файлу загрузки\atolimp.flg;
   7) Служебный файл-флаг для выгрузки - путь к файлу выгрузки\atolexp.flg.
5. В PAPYRUS при настройке кассового узла (Справочники/Оборудование/Кассы):
   1) Файлы импорта - путь к файлу выгрузки;
   2) Пути экспорта - путь к файлу загрузки;
   3) Логические номера касс - номера ПК из настроек РМК на кассах
      через запятую (они должны быть разными);
   4) Флаг "Передавать контрольную цифру кода" установить.


Настройки АТОЛ-CARD для совместной работы с PAPYRUS
---------------------------------------------------
1. В PAPYRUS открыть сессию (Оборудование/Кассовый узел/Открыть сессию).
В АТОЛ-CARD будут загружены дисконтные карты и созданы категории клиентов
"PAPYRUS - скидка n%".
2. В администраторе АТОЛ-CARD создать схему для работы с PAPYRUS с кодом 1001.
В схеме создать скидки в соответствии с категориями и установить процент скидки.
Для каждой скидки добавить фактор "Категория клиента" и выбрать
соответствующую категорию.
ПРИМЕР:
- после загрузки карт в АТОЛ_CARD появилась категория "PAPYRUS - скидка 3.00%";
- создаем схему PAPYRUS  с кодом 1001;
- в схему добавляем скидку "Скидка 3%", на закладке "Объект" устанавливаем:
   1) Тип - процентная скидка;
   2) Значение - -3%, область действия - на стоимость;
   3) Алгоритм - без алгоритма;
- в скидку добавляем фактор "Категория клиента" с названием "Скидка 3%",
  в факторе выбираем категорию "PAPYRUS - скидка 3.00%".
Аналогично для других категорий.
3. В папке "Настройки системы" - "Параметры" на закладке "Регистрация"
убрать флаг "Разрешить автоматическую регистрацию клиентов".
4. В РМК АТОЛ в настройках на закладке "Скидки/надбавки" - "Внешн.автоматические":
   1) Флаг "Использовать АТОЛ-CARD" установить.
   2) Схемы по умолчанию - схема PAPYRUS (код 1001).
