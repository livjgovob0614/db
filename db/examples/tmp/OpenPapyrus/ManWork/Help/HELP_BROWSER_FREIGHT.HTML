Таблица отчета по фрахтам



Фрахт

Примерный вид таблицы отчета показан на рис..
Таблица содержит следующие колонки:
Отгружен Если документ имеет признак отгружен, то в этой колонке виден символ $X$.
Дата документа
Номер документа
Контрагент Наименование контрагента по документу.
Дата доставки Дата доставки, указанная в данных по фрахту.
Транспорт Наименование транспортного средства из данных по фрахту документа.
Город назначения Город назначения из данных по фрахту документа.
Адрес доставки Текст адреса доставки из данных по фрахту документа.
Сумма Номинальная сумма документа.
Брутто Суммарная масса брутто в килограммах всех номенклатурных позиций, входящих в документ.
Объем Суммарный грузовой объем в кубических метрах всех номенклатурных позиций, входящих в документ.
Мест Суммарное количество упаковок.

Действия в таблице


Документ Enter Открывает диалог редактирования документа, которому соответствует текущая строка отчета. Переключить признак отгрузки F8 Изменяет значение признака Отгружен документа (то есть, если признак отсутствовал, то устанавливает его, если же он был включен, то - снимает. Итоги F9 Открывает диалог итоговых значений отчета. Экспорт Ctrl-R Позволяет экспортировать все документы, представленные в отчете в соответствии с выбранной конфигурацией экспорта. Подробно об экспорте документов см. стр.. Печать F7 Фильтр Ctrl-V Позволяет изменить критерии фильтрации отчета.
